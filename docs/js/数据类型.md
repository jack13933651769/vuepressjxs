## 数值表达及转换

十进制: 直接写<br>

八进制:以0 开头.逢八进一 超过或者等于8 当作十进制处理<br>

十六进制: 0x开头 0-9及a-f<br>

浮点数: 带小数点的数值   非常大或者非常小的 可以用科学技术法表示

```js

let floatNum = 3.125e7; // 等于 31250000

//整数 正e  极小数使用-e
```

<br>

NaN  不是数值

有isNaN 能转换成数字的都是false  不能转换的就是true
```js

consolse.log(NaN==NaN)  //false
```
